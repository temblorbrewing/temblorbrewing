# Temblor Brewing Company — Website

Static website for Temblor Brewing Company, deployed via Netlify.

## Pages

| File | URL | Description |
|------|-----|-------------|
| `index.html` | `/` | Main homepage |
| `menu.html` | `/menu.html` | Food, Tap & Drinks menu |
| `private-events.html` | `/private-events.html` | Private events booking page |
| `careers.html` | `/careers.html` | Join Our Team / Job Application |

## Deployment

1. Push this repository to GitHub
2. Connect the GitHub repo to Netlify
3. Netlify auto-deploys on every push to `main`

No build step required — all HTML is self-contained.
